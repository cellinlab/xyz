### type对应的类别

> 响应的数据中很多带有 type，以下是各个 type 所对应的类别

| type                   | 说明       |
| :--------------------- | :--------- |
| PODCAST                | 节目       |
| EPISODE                | 单集       |
| USER                   | 用户       |
| COMMENT                | 评论       |
| wechatOfficialAccounts | 微信公众号 |
| wechat                 | 微信号     |
| jike                   | 即刻主页   |
| email                  | 邮箱       |
| weibo                  | 新浪微博   |
